Описание расписания НК


Расписание - в НК встроена работа по расписанию на неделю. Возможна установка изменений с точностью до 1 часа.
Работа расписания базируется на профилях, фактически расписание это таблица применения профилей по часам и дням недели. Для эффективного использования расписаний необходимо в начале создать требуемые профили. Помимо переключения между профилями возможна постановка ТН на паузу (ТН выключен) и добавка к целевой температуре(±Температура). Все варианты действий по расписанию отображены на рисунке (с номерами это профиля).

Временно можно выбрать любое расписание, которое будет действовать до прихода времени очередного. Если все таки нужно временно использовать заданные вручную настройки и чтобы сохранить автоматизм - можно выбрать расписание, в котором установлено на все время "± Температура" равная 0.
Работа по расписанию - включение работы по расписанию НК. При включении этой опции на панели управления над списком профилей надпись 'ПРОФИЛЬ' меняется на 'РАСПИСАНИЕ'.
Текущее расписание - выбор текущего расписания (выпадающий список). НК содержит до 9 независимых расписаний.
Редактировать имя - имя расписания можно изменить использую это поле. Сохранение происходит по кнопке 'Сохранить'.
Автовыбор расписания (0 - нет) - месяц и декада переключения каждого расписания. Для этого галочка "Работа по расписанию" должна быть включена. Данная опция служит для переключения между расписаниями
Установить профиль - выбор действия при создании расписания (варианты действий см рисунок).
Таблица текущего расписания - таблица в котором показано и можно менять текущее расписание. Порядок создания/изменения расписания:
- выбрать желаемое действие использую поле Установить профиль;
- нажав левую кнопку мыши выделить на клетках таблицы расписания время выбранного действия;
- для стирания клеток расписания нажать Shift + левая кнопка мыши и выделить стираемые клетки.
Чтобы текущий профиль не менялся нужно заполнить ± температурой равной 0.
